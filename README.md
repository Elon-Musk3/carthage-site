# Carthage Site Final

## Arquivos principais
- `index.html` — página principal.
- `style.css` — estilos, responsividade, animações e hover effects.
- `script.js` — loader, reveal on scroll, menu mobile e efeitos leves.
- `assets/carthage-logo.png` — logomarca usada no topo, rodapé e favicon.
- `legal/` — páginas legais com botão de voltar.
