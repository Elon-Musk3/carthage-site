# Vídeos promocionais

Nenhum vídeo ou áudio foi inventado neste pacote.

Para ativar o player da página `aplicativo.html`:

1. coloque arquivos autorizados nesta pasta;
2. prefira MP4 H.264 e, quando disponível, WebM;
3. comprima versões específicas para site e mobile;
4. adicione legendas;
5. configure os caminhos em `js/config.js`, dentro de `MEDIA_CONFIG`.

O site não reproduz áudio automaticamente. Use apenas trilha criada para a Carthage, licenciada para uso comercial ou fornecida com autorização.
